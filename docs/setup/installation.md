# Installation Guide

## Requirements

* Java 17+
* PostgreSQL
* Gradle or Maven

---

## Steps

1. Clone repository

```id="d2y0ml"
git clone <your-repo-url>
```

2. Navigate to project

```id="5ex9o9"
cd BMS_V_5
```

3. Run backend

```id="n9r3nt"
./gradlew bootRun
```
