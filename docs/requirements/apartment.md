# Apartment Requirements

## Overview

Represents a unit inside a building.

---

## Rules

* Apartment must belong to a building
* Apartment name must be unique within a building

---

## Permissions

* SUPER_ADMIN / ADMIN → create, update, view

---

## Data

* Name
* Floor number
* Sector/block
* Room details
* Area and parking

---

## Validation

* Duplicate apartment name in same building → rejected
