# Features

## Building Management

* Create and manage buildings
* Store address and infrastructure details

---

## Apartment Management

* Manage apartments within buildings
* Track rooms, area, and facilities

---

## Ownership Management

* Multiple owners per apartment
* Percentage-based ownership tracking
* Ownership history support

---

## Lease Management

* One active lease per apartment
* Flexible resident assignment
* Lease lifecycle tracking

---

## User Management

* Create admins, owners, residents
* Manage user profiles

---

## Security

* JWT-based authentication
* Role-based access control
